# UI/UX Audit (Admin + Customer PWA)

기준일: 2026-04-27  
범위: Admin 콘솔, Customer PWA, 루트 라우팅  
목표: 동작 검증 기반 IA 정리 (시각 꾸미기 아님)

## Audit 기준

- 1) 화면 목적이 단일한가
- 2) 주요 행동이 명확한가
- 3) 상태/데이터 소스가 명확한가
- 4) 명령 실행 후 피드백이 있는가
- 5) 로딩/빈/오류 상태가 있는가
- 6) 중복/죽은 UI가 없는가
- 7) 한국어 서비스 문구 중심인가
- 8) Customer 데이터 경계가 안전한가
